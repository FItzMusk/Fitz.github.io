2. 金属切削原理与刀具

2.1 主运动、进给运动

2.2 Cutting parameters

  • Cutting speed $v$ $v=\frac{\pi d_w n}{1000\times 60}$
  • Feeding rate $f$ $v_f=nf=nzf_z$
  • Back cutting depth $a_p$ $a_p=\frac{d_w-d_m}{2}$（待加工、已加工）

2.3 Tool

前面Top、主后刀面principal flank surface、副后刀面auxiliary flank surface、主切削刃principal cutting edge、副切削刃auxiliary cutting edge、刀尖Nose

2.4 参考平面 & 标注角度

切削平面tool cutting edge plane $P_s$ 、基面 reference plane $P_r $、正交平面orthogonal plane $ P_o $，前角rake angle $ \gamma _o $、
后角 flank angle $\alpha_o $、主偏角tool cutting edge angle $\kappa_r$、副偏角tool minor cutting edge angle $\kappa_r’$、刃倾角cutting edge inclination angle $\lambda_s$。

2.5 刀具材料Material

  • 碳素工具钢carbon tool steel

  • 合金工具钢alloy tool steel

  • 高速钢high speed steel
  • 硬质合金钢hard alloy steel
    • 红 K: YG 钨钴WCo
    • 黄 M: YW 钨W
    • 蓝 P: YT 钨钴钛WCoTi
    • 陶瓷、金刚石

2.6 切屑Chip

  • 变形区

  • 切削类型Type

  • 积屑瘤Built-up edge like tumour
    • 塑性、切削速度不高、形成连续切屑
    • 高速、低速宽刀刃、增大前角、换材料

2.7 切削力Cutting force

克服塑性变形、弹性变形、摩擦力，主切削力main cutting force$ F_c$、进给力feeding force $F_f$、切深抗力 $F_p$。

2.8 寿命Life

磨损abrasion、破损breakage (脆性、塑性) VB(作为刀具磨钝标准)、刀具寿命 (总寿命)

5. Quality analysis and control

5.1 加工精度Precision

形状精度、位置精度、尺寸精度 shape, position, size

5.2 工艺系统

机床、夹具、刀具、工件

  • 因：工艺系统误差 (so call 原始误差)
  • 果：加工误差

5.3 原始误差Original error

5.4 机床误差Machine tool error

  • slide way导轨误差: $a:\Delta R\approx \frac{\sigma_z^2}{2R}$ ，$b: \Delta R=\sigma _y$

    加工表面法向为Y，切向为Z，Y为敏感方向

  • spindle 主轴误差：纯轴向运动 $\Delta x$，纯径向移动 $\Delta r$，纯角度摆动 $\Delta \gamma$

    提高精度：高精度轴承、高装配精度、高速主轴部件平衡，轴承
    预紧。

  • transmission chain传动链误差：$\Delta \phi_{1n} =K_1\Delta \phi_1$， $K_q=\frac{1}{24}i_{差}i_{分}$$
    提高精度：缩短传动链、降速传动、提高传动件制造和装夹精度、
    传动误差补偿装置

5.5 工艺系统受力变形对加工精度影响Deformation

Stiffness刚度：$k = \frac{F_y}{y}$

  • Point受力点位置变化：

    1. 若刚度很高

    2. 刚度很低

    3. 总：$y = y_{系统} + y_{工件}$

  • 切削力变化–误差复映error reflection

$\Delta_{工件} = y_1 − y_2 =\frac{C}{K_{系统}}(a_{p1} − a_{p2})$,

$\Delta_{毛坯} = a_{p1} − a_{p2}$,
$\Delta_{工件}=\frac{C}{K_{系统}}\Delta_{毛坯}$,
所以, 误差复映系数 ($\epsilon<1$ ): $\epsilon_总 = \epsilon_1\epsilon_2\epsilon_3 \dots$

$\epsilon = \frac{\Delta_{工件}}{\Delta_{毛坯}} =\frac{C}{K_{系统}}$

5.6 加工误差综合分析Manufacturing error analysis

  • System 系统性误差：constant常值系统误差、variable变值系统误差

  • random随机误差：正态分布gaussian normal

  • qualified area合格区域：$6\sigma$，公差带 $T \ge 6\sigma$

  • craft coefficient:
    $C_{pk}=C_p-\frac{E}{3\sigma}=\frac{T}{6\sigma}-\frac{E}{3\sigma}$

5.7 Surface quality

  • 几何形状特征：表面粗糙度、表面波度
  • 物理化学性能变化：冷作硬化、金相组织变化、残余应力

6. 工艺规程设计procedures

6.1 生产类型 Types

单件生产、小批量生产、大批量生产

$$N = Q_n(1 + a)(1 + b)$$

N 产品年产 n 每台零件 a 备品率 b 废品率

6.2 制造系统三流

物质流matter flow、信息流、资金流money flow

6.3 基准

  • 精基准：基准重合、统一基准、互为基准、自为基准
  • 粗基准：重要表面、自为基准、便于装夹、只用一次

6.4 加工余量working allowance

  • 单边最小余量
    $$Z_{bmin} = T_a + R_{ya} + H_a + |\vec{e_a} + \vec{\epsilon_b}|$$

  • 双边最小余量（外圆、内孔）
    $$2Z_{bmin} = T_a + 2(R_{ya} + H_a + |\vec{e_a} +\vec{\epsilon_b}|)$$
    双顶尖外圆$$\Rightarrow T_a + 2(R_{ya} + H_a + |\vec{e_a}|)$$
    浮动镗孔/研磨打磨外圆内孔$$\Rightarrow T_a + 2(R_{ya} + H_a)$$

6.5 尺寸链dimension chain

定义：相互联系connected、按一定顺序order、构成封闭形式closed的一组尺寸（相关尺寸）
封闭环closed component、组成环partial（增环increasing component、减环decreasing）
